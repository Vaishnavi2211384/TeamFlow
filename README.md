# TaskFlow - Team Task Manager

TaskFlow is a simple web application that helps teams manage projects, assign tasks, and track progress in an organized way. It includes role-based access so admins and members can have different permissions.

## 🚀 Features

* User Authentication (Signup / Login)
* Create and manage projects
* Assign tasks to team members
* Update task status (To Do, In Progress, Done)
* Due date tracking with overdue highlighting
* Role-based access (Admin / Member)
* Dashboard to view assigned tasks

## 🛠️ Tech Stack

* Frontend: React (Vite)
* Backend & Database: Supabase
* Styling: Tailwind CSS
* Deployment: Railway

## ⚙️ How to Run Locally

1. Clone the repository:
   git clone https://github.com/Vaishnavi2211384/team-flow.git

2. Navigate to the project folder:
   cd team-flow

3. Install dependencies:
   npm install

4. Add environment variables:
   Create a `.env` file and add:
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key

5. Run the project:
   npm run dev
